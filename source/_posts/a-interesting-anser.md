### 程序语言编程
```c++
if (a < b) min = a; else min = b;
```

<!--more-->

### 英语编程：
```
If a is less than b, set min to a, otherwise set min to b. 
```
### 中文编程
```
如果a小于b，那么给 min 赋值 a，否则给 min 赋值 b。

```

### 民科追捧的“中文编程”
```
如果 (甲 < 乙) 最小值 = 甲；否则 最小值 = 乙；
```
